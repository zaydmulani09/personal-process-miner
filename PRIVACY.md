# Personal Process Miner — Privacy Policy

Last updated: 2026-05-21

## What we collect
Personal Process Miner runs entirely on your device. All data — keystrokes, mouse events, window titles, screenshots, and automation scripts — is stored locally in a SQLite database on your machine.

## What we do NOT collect
- No data is sent to any server operated by the developers
- No telemetry, analytics, or crash reporting
- No account creation or login required

## AI Vision (optional)
If you configure an AI Vision provider (Claude, OpenAI, Groq, Gemini, or Grok), screenshots are sent to that provider's API when you trigger an automation. This is opt-in and only occurs when you explicitly run a vision action. Review each provider's privacy policy for how they handle image data.

## API keys
API keys you enter are stored locally in the SQLite database on your device. They are never transmitted to the app developers.

## Your control
You can delete all captured data at any time using the Privacy Controls page in the app. Uninstalling the app removes all local data.

## Contact
This is an open source project. For questions, open an issue on GitHub.
